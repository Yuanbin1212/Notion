1. 编程语言介绍
    1. 分类：
        1. 机器语言
        2. 汇编语言
        3. 高级语言（编译型、解释型号）
    2. 总结：
        1. 执行效率：机器语言>汇编语言>高级语言（编译型>解释型）
        2. 开发效率：机器语言<汇编语言<高级语言（编译型<解释型）
        3. 跨平台性：解释型具有极强的跨平台型

5. 注释是对关键代码的解释说明
    1. 单行注释：#
                # 这是一行xxx的代码
                print("hello")  # 这是一行xxx的代码
    2. 多行注释：''''''  """"""
                """
                笔记
                """
    3. 被注释的代码不会被执行
